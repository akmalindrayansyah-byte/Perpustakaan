 # Sistem Perpustakaan
 # Keterangan Umum Aplikasi
 Aplikasi Perpustakaan Berbasis Web adalah sebuah sistem informasi terkomputerisasi yang dirancang untuk mengelola seluruh operasional perpustakaan secara digital. Sistem ini memfasilitasi pengelolaan data buku, proses peminjaman, pengembalian, dan perhitungan denda secara efisien dan terstruktur.
 ---
# 1. Halaman Dashboard
<img width="1914" height="913" alt="Screenshot 2025-12-31 112938" src="https://github.com/user-attachments/assets/4ad12752-8e9e-4704-81b5-71155bfd6757" />
Halaman Dashboard  menampilkan ringkasan informasi sistem, seperti jumlah buku, jumlah siswa, serta data transaksi peminjaman dan pengembalian. Dari halaman ini, admin dapat mengakses seluruh menu pengelolaan.

# 2. Tambah Buku
<img width="1904" height="916" alt="Screenshot 2025-12-31 120904" src="https://github.com/user-attachments/assets/ef2b99ab-8e81-443e-b1ea-83c5b56effd9" />
Halaman Tambah buku untuk menampilkan daftar buku yang tersedia di perpustakaan. Admin dapat menambah, mengubah, menghapus, dan mencari data buku berdasarkan judul atau kategori.

# 3. Peminjaman
<img width="1919" height="905" alt="Screenshot 2025-12-31 121503" src="https://github.com/user-attachments/assets/4d6c089f-cf90-48d8-86bf-e20d333a96ad" />
Halaman Peminjaman digunakan untuk melakukan transaksi peminjaman buku oleh anggota perpustakaan. Petugas dapat memproses peminjaman dengan memilih anggota yang valid dan buku yang tersedia, serta mencatat tanggal peminjaman untuk keperluan tracking dan pengembalian.

# 4. Pengembalian
<img width="1911" height="913" alt="image" src="https://github.com/user-attachments/assets/5117ff46-be0e-469b-852f-55e9ed69f7d8" />
Halaman Pengembalian digunakan untuk memproses pengembalian buku yang telah dipinjam. Petugas dapat memasukkan ID peminjaman untuk mencatat pengembalian, menghitung denda keterlambatan jika ada, dan mengupdate status ketersediaan buku di sistem.

# 5. Laporan
<img width="1908" height="911" alt="image" src="https://github.com/user-attachments/assets/7fd4f0cc-0cec-43ec-9f5f-5b2166db7e7d" />
Halaman Laporan digunakan untuk menampilkan seluruh data transaksi peminjaman buku dalam sistem. Petugas dapat melihat riwayat lengkap peminjaman, status pengembalian, dan informasi denda untuk keperluan monitoring dan pelaporan aktivitas perpustakaan.

 # 6. Anggota
 <img width="1911" height="912" alt="image" src="https://github.com/user-attachments/assets/54b1c103-a84c-4d23-a58b-5fd1cf34980e" />
Halaman Anggota digunakan untuk mengelola data anggota perpustakaan. Admin dapat menambahkan anggota baru dengan mengisi informasi lengkap, serta melihat, mengedit, atau menghapus data anggota yang sudah terdaftar dalam sistem.

# Kesimpulan 
Dengan adanya aplikasi perpustakaan berbasis web ini, proses pengelolaan perpustakaan menjadi lebih efektif, terstruktur, dan mudah digunakan baik oleh admin maupun masyarakat umum ataupun siswa.
